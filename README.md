## ✨ Overview

The **IRCTC Management System** is a console-based application that simulates a real-world railway reservation platform.
It automates ticket booking, cancellation, and train management using **Core Java and MySQL**, ensuring efficient and real-time data handling.

---

## 🚀 Features

* 🔐 Secure User Registration & Login
* 🔍 Train Search (Source, Destination, Date)
* 🎟️ Real-Time Ticket Booking System
* ❌ Ticket Cancellation with Seat Update
* 📄 Booking History Tracking
* 🆔 Unique PNR Generation
* 🛠️ Admin Controls for Train Management
* ⚠️ Input Validation & Error Handling

## ⚙️ System Workflow

```text
User Registration/Login
        ↓
 Search Trains
        ↓
 Book Ticket (Seat Allocation)
        ↓
 Generate PNR
        ↓
 View / Cancel Booking
```
